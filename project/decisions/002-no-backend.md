# 002: バックエンド非連携（モックデータのみ）

## 日付
2025-10-01

## ステータス
採用

## 決定
フロントエンドはモックデータのみで動作させ、バックエンドAPI連携は行わない

## 背景
まずUI/UXの検証と画面実装を優先したい。

## 理由
- **高速なプロトタイピング**: API待ちなく画面開発に集中できる
- **独立した開発**: バックエンド開発と並行して進められる
- **仕様確定**: 画面を見ながら仕様を詰められる

## 制約（禁止事項）
- fetch / axios によるHTTP通信
- WebSocket / Server-Sent Events
- localStorage / sessionStorage / IndexedDB（永続化）
- 実際の認証処理（JWT / OAuth）

## 影響
- すべてのデータは `mock.ts` ファイルで管理
- 状態変更はローカルステートのみ（リロードでリセット）
- 将来のAPI連携時はカスタムフック層で吸収
