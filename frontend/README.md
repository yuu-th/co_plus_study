# CO+ Study - Frontend

学習意欲のある小中学生向けWebアプリケーションのフロントエンド実装。

## 🎯 プロジェクト概要

このプロジェクトは、**描画・UI表示に特化**したReactアプリケーションです。
バックエンドAPI連携やデータ永続化は含まず、モックデータを用いた静的な画面表示とインタラクションを実現しています。

## 🚀 セットアップ

### 前提条件

- Node.js 18.x 以上
- npm または yarn

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` にアクセスしてください。

### ビルド

```bash
npm run build
```

### プレビュー

```bash
npm run preview
```

## 🛠️ 技術スタック

- **React** 18.x - UIライブラリ
- **TypeScript** - 型安全な開発
- **Vite** - 高速なビルドツール
- **React Router** v6 - ルーティング

## 📁 プロジェクト構造

```
src/
├── assets/           # 静的アセット
├── components/       # コンポーネント
│   ├── common/      # 共通コンポーネント
│   ├── layout/      # レイアウトコンポーネント
│   └── archive/     # ARCHIVE関連コンポーネント
├── pages/           # ページコンポーネント
├── types/           # TypeScript型定義
├── mockData/        # モックデータ
├── styles/          # グローバルスタイル
├── router.tsx       # ルーティング設定
├── App.tsx          # アプリケーションルート
└── main.tsx         # エントリーポイント
```

## 🎨 主要機能

### ARCHIVEページ（メイン実装）

イメージ図に基づいて実装された、学習記録の可視化ページ:

#### カレンダータブ
- **連続ログイン日数**: 953日を大きく表示（カウントアップアニメーション付き）
- **月次カレンダー**: 3ヶ月分（9月、8月、7月）を縦並び表示
- **活動日の色分け**: ログイン/日報/両方を異なる色で表示
- **ウィークリー表示**: S M T W T F S 形式で直近1週間の活動を表示

#### スキルタブ
- **バッジグリッド**: 2列レイアウトで獲得したバッジを表示
- **金・銀・銅のメダル**: バッジタイプに応じた視覚的な表現
- **獲得条件**: 各バッジの説明と獲得条件を表示

### その他のページ

- **ホームページ**: 各機能へのナビゲーション
- **学習日報ページ**: （簡易実装）

## 📱 レスポンシブ対応

- **デスクトップ**: 1024px以上 - 全機能を広々と表示
- **タブレット**: 768px - 1023px - 適切なレイアウト調整
- **モバイル**: 640px以下 - タッチ操作に最適化

## 🎨 デザインシステム

### カラースキーム

- **基調色**: ライトブルー（#E8F4F8）と白
- **アクセントカラー**: 
  - ブルー（#4DB8E8）
  - オレンジ（#FF9F40）
  - ゴールド（#FFD700）
- **バッジカラー**: 金・銀・銅・プラチナ

### タイポグラフィ

- **フォント**: ヒラギノ角ゴ / メイリオ
- **サイズ**: 12px - 72px（継続日数は特大）

## 🚫 実装していない機能

以下の機能は**スコープ外**です:

- バックエンドAPI連携
- 実際の認証処理
- データベース連携
- 状態の永続化（localStorage等）
- リアルタイムチャット機能
- ファイルアップロード

## 📝 モックデータ

すべてのデータは `src/mockData/` に配置されています:

- `users.ts` - ユーザーデータ
- `calendar.ts` - カレンダー・活動記録（953日の連続ログイン）
- `badges.ts` - バッジデータ（金銀銅）
- `diaries.ts` - 学習日報データ
- `chats.ts` - チャットデータ
